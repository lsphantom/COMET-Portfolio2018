SIMULATION: NOAA/NWS

Predicting Convective Cessation for Aviation Forecasters

About this project

COMET develops effective simulations that allow learners to make adjustments in a safe environment. The case simulation in this lesson asks learners to decide if and when a TAF should be amended. It features a timed release of products to mimic some of the pressure aviation forecasters experience while working. Occasional distractions have intentionally been included to model the interruptions learners might encounter on the forecast desk.
